# 📅 Meeting Scheduler

A modern, professional meeting scheduler web application with real-time notifications, elegant animations, and persistent storage.

## ✨ Features

- **Create & Schedule Meetings** - Add meetings with title, date, time, and duration
- **Real-Time Notifications** - Get alerted 5 minutes before each meeting
- **Multiple Alert Types**:
  - 🔔 In-app popup notifications
  - 🖥️ System/Desktop notifications
  - 🔊 Audio alerts (triple beep)
  - 📳 Mobile vibration feedback
- **Cross-Window Alerts** - Notifications appear even if you're working in another window
- **Persistent Storage** - Meetings are saved using localStorage
- **Countdown Timer** - See how long until each meeting starts
- **Delete Meetings** - Remove meetings with a single click
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Professional Animations** - Smooth, bouncy, enterprise-quality animations throughout

## 🎨 Design Highlights

- **Glassmorphism UI** - Modern frosted glass effect with backdrop blur
- **Animated Gradients** - Dynamic background with smooth color transitions
- **Floating Blob Effects** - Sophisticated multi-layer blob animations
- **Cubic-Bezier Timing** - Professional animation easing functions
- **Color Palette** - Purple, pink, and blue gradient theme

## 🚀 Getting Started

2. Open `index.html` in your web browser
3. Start scheduling meetings!


## 📱 Usage

1. **Add a Meeting**:
   - Fill in the meeting title
   - Select the date
   - Enter the time
   - Specify duration (in minutes)
   - Click "Schedule Meeting"

2. **View Meetings**:
   - All scheduled meetings appear below the form
   - See countdown timers showing when meetings start

3. **Remove Meetings**:
   - Click the ✕ button on any meeting card to delete it

4. **Get Notified**:
   - Notifications trigger automatically 5 minutes before each meeting
   - Check browser notifications are enabled for best experience
